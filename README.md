## SENSOR MODULE CONNECTION

```
Maxbotix connection

Sensor Module pin       Maxbotix pin        Wire color
------------------------------------------------------
A (pulse in)            2 (pulse out)       Yellow
GND                     GND                 Black
VCC                     V+                  Red
```
